# Claude 中文包安装详细步骤

## 🎯 快速安装（推荐方法）

### 第一步：找到 Claude 应用目录

**Windows 系统：**
1. 按 `Win + R` 打开运行对话框
2. 输入：`%LOCALAPPDATA%\Claude`
3. 点击确定

**macOS 系统：**
1. 打开访达（Finder）
2. 按 `Cmd + Shift + G`
3. 输入：`/Applications/Claude.app/Contents/Resources/`

### 第二步：找到语言文件夹

在打开的目录中，找到以下路径：
- Windows: `resources\app\locales\`
- macOS: `app\locales\`

**如果没有 `locales` 文件夹：**
- 右键创建新文件夹，命名为 `locales`

### 第三步：复制语言包文件

1. 找到我为你创建的 `zh-CN.json` 文件
2. 复制这个文件到上一步找到的 `locales` 文件夹中

### 第四步：重启 Claude

完全关闭 Claude 应用，然后重新打开

### 第五步：切换语言

1. 打开 Claude 应用
2. 点击菜单栏的设置（Settings）
3. 找到语言（Language）选项
4. 选择"中文（简体）"或"zh-CN"
5. 重启应用使更改生效

## ⚠️ 可能遇到的问题

### 问题 1：找不到 locales 文件夹

**解决方案：**

Claude 桌面应用可能使用 Electron 的 asar 打包格式，需要解包：

1. 安装 asar 工具：
   ```bash
   npm install -g asar
   ```

2. 找到 `app.asar` 文件：
   - Windows: `%LOCALAPPDATA%\Claude\resources\app.asar`
   - macOS: `/Applications/Claude.app/Contents/Resources/app.asar`

3. 解包：
   ```bash
   # Windows (在 PowerShell 中运行)
   cd %LOCALAPPDATA%\Claude\resources
   asar extract app.asar app
   
   # macOS (在终端中运行)
   cd /Applications/Claude.app/Contents/Resources/
   asar extract app.asar app
   ```

4. 在解包后的 `app` 文件夹中创建 `locales` 文件夹

5. 复制 `zh-CN.json` 到 `app/locales/`

6. 重新打包：
   ```bash
   # 备份原文件
   mv app.asar app.asar.backup
   
   # 重新打包
   asar pack app app.asar
   ```

### 问题 2：没有权限修改文件

**Windows 解决方案：**
1. 右键点击 Claude 应用文件夹
2. 选择"属性" → "安全"
3. 点击"编辑"，给你的用户账户完全控制权限

**macOS 解决方案：**
```bash
sudo chmod -R 755 /Applications/Claude.app/Contents/Resources/
```

### 问题 3：修改后应用无法启动

**解决方案：**
1. 删除修改的文件
2. 如果做了备份，恢复 `app.asar.backup`
3. 重新安装 Claude 应用

## 🔧 高级方法：修改配置文件

如果上述方法不起作用，可以尝试修改配置文件：

### Windows:
1. 打开：`%APPDATA%\Claude\config.json`
2. 如果文件不存在，创建它
3. 添加以下内容：
   ```json
   {
     "language": "zh-CN",
     "locale": "zh-CN"
   }
   ```

### macOS:
1. 打开：`~/Library/Application Support/Claude/config.json`
2. 如果文件不存在，创建它
3. 添加相同的配置内容

## 📝 验证安装成功

安装成功后，你应该看到：

✅ 菜单栏显示：文件、编辑、视图、窗口、帮助  
✅ 输入框提示：向 Claude 发送消息...  
✅ 按钮显示：发送、停止、重新生成  
✅ 设置界面全部为中文  

## 🆘 需要帮助？

如果以上步骤都不起作用：

1. **检查 Claude 版本**：确保你使用的是桌面版 Claude（Electron 应用）
2. **查看应用日志**：可能会有错误信息
3. **尝试重新安装**：完全卸载后重新安装 Claude
4. **联系支持**：如果是官方版本，可能需要等待官方支持中文

## 💡 提示

- 每次 Claude 更新后，可能需要重新安装语言包
- 建议保存好 `zh-CN.json` 文件以便后续使用
- 修改系统文件前务必备份
