# Claude 桌面应用中文语言包

这是一个为 Claude 桌面应用程序（Electron 版本）制作的完整中文翻译包，包含界面元素、帮助文档和使用说明。

## 📦 包含内容

- `zh-CN.json` - 完整的中文翻译文件
- `README.md` - 本安装说明文档

## 🎯 翻译范围

本语言包涵盖以下模块的完整翻译：

- **应用核心**：菜单、设置、通知
- **对话功能**：聊天界面、历史记录、消息操作
- **协作模式**：文件管理、权限控制、任务状态
- **文件操作**：上传、下载、附件管理
- **插件系统**：插件管理、安装、配置
- **技能系统**：技能创建、编辑、管理
- **任务管理**：任务创建、状态跟踪、计划任务
- **浏览器集成**：网页导航、截图、自动化
- **错误提示**：各类错误信息和解决建议
- **快捷键**：键盘快捷键说明

## 📥 安装方法

### 方法一：手动安装（推荐）

1. **找到 Claude 应用的语言文件目录**

   Windows 路径通常为：
   ```
   C:\Users\[用户名]\AppData\Local\Claude\resources\app\locales\
   ```

   macOS 路径通常为：
   ```
   /Applications/Claude.app/Contents/Resources/app/locales/
   ```

2. **备份原有文件**（可选但推荐）
   
   如果目录中已有语言文件，建议先备份

3. **复制语言包文件**
   
   将 `zh-CN.json` 复制到上述 `locales` 目录中

4. **重启 Claude 应用**

5. **切换语言**
   
   打开 Claude → 设置 → 语言 → 选择"中文（简体）"

### 方法二：开发者模式安装

如果你熟悉 Electron 应用开发，可以通过以下方式安装：

1. 解压 Claude 应用的 `app.asar` 文件
2. 将 `zh-CN.json` 放入解压后的 `locales` 目录
3. 重新打包 `app.asar`
4. 重启应用

**工具推荐**：
- Windows: [asar](https://github.com/electron/asar)
- macOS: `npm install -g asar`

```bash
# 解包
asar extract app.asar app

# 复制语言文件
cp zh-CN.json app/locales/

# 重新打包
asar pack app app.asar
```

## 🔧 配置说明

### 修改应用配置文件

某些版本的 Claude 可能需要手动修改配置文件来启用中文：

1. 找到配置文件（通常在用户数据目录）：
   - Windows: `%APPDATA%\Claude\config.json`
   - macOS: `~/Library/Application Support/Claude/config.json`

2. 添加或修改语言设置：
   ```json
   {
     "language": "zh-CN"
   }
   ```

3. 保存并重启应用

## ✅ 验证安装

安装成功后，你应该能看到：

- 菜单栏显示为中文（文件、编辑、视图等）
- 设置界面显示为中文
- 对话框和提示信息显示为中文
- 按钮和标签显示为中文

## 🐛 故障排除

### 问题：重启后仍显示英文

**解决方案**：
1. 确认 `zh-CN.json` 文件位置正确
2. 检查文件权限，确保应用可以读取
3. 尝试完全卸载并重新安装 Claude
4. 检查配置文件中的语言设置

### 问题：部分界面未翻译

**解决方案**：
1. 这可能是因为该版本的 Claude 使用了不同的翻译键
2. 可以提交 Issue 反馈具体未翻译的部分
3. 等待语言包更新

### 问题：翻译文件格式错误

**解决方案**：
1. 确保 `zh-CN.json` 是有效的 JSON 格式
2. 使用 JSON 验证工具检查文件
3. 重新下载原始语言包文件

## 📝 自定义翻译

如果你想修改某些翻译，可以直接编辑 `zh-CN.json` 文件：

1. 使用文本编辑器打开 `zh-CN.json`
2. 找到要修改的键值对
3. 修改对应的中文翻译
4. 保存文件并重启 Claude

**示例**：
```json
{
  "chat": {
    "placeholder": "向 Claude 发送消息..."  // 可以改为你喜欢的文本
  }
}
```

## 🔄 更新说明

当 Claude 应用更新后，可能会添加新的界面元素。如果发现新功能未翻译：

1. 检查是否有新版本的语言包
2. 下载最新的 `zh-CN.json` 并替换旧文件
3. 重启应用
